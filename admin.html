⚙️ Firebase no está configurado

Sigue estos pasos una sola vez (≈5 minutos) para conectar la base de datos.

1
Ve a console.firebase.google.com y crea un proyecto llamado GloboBYM.
2
En el panel del proyecto, haz clic en el ícono </> (Web) → apodo globobym-web → Registrar.
3
Copia los valores de firebaseConfig y pégalos en el archivo firebase-config.js.
4
En el menú lateral → Firestore Database → Crear base de datos → Modo producción → Reglas → pega:
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /pedidos/{id} {
      allow create: true;
      allow read, update, delete: if request.auth != null;
    }
  }
}
5
En el menú lateral → Authentication → Comenzar → Email/contraseña → Habilitar.
6
Authentication → Usuarios → Agregar usuario con tu email y contraseña. ¡Listo!
